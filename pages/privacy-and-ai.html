Personvern ved bruk av KI

Personvern ved bruk av KI handler først og fremst om å tenke gjennom hva du deler. KI kan være nyttig i studiearbeid, men du bør unngå å legge inn personopplysninger, sensitive detaljer eller materiale som ikke er ment for deling.

Hvorfor personvern er viktig ved bruk av KI

Selv om KI kan være et nyttig verktøy til bruk i utdanningssammenheng, innebærer det også en del risiko å bruke det. Dette er spesielt knyttet til personvern og KI sin behandling av personopplysninger.

Flere generative KI-tjenester kan lagre og bruke data som legges inn for å videre trene seg selv, og dersom disse dataene inneholder personopplysninger eller sensitive data, kan dette få alvorlige konsekvenser.

Hva regnes som personopplysninger i KI-sammenheng?

Ifølge GDPR artikkel 4 er en personopplysning enhver opplysning som direkte eller indirekte kan knyttes til en identifiserbar, fysisk person. Dette inkluderer navn, ID-nummer, lokasjonsdata, IP-adresser, eller faktorer som fysisk, fysiologisk, genetisk, mental, økonomisk, kulturell eller sosial identitet.

En personopplysning er med andre ord mye mer enn navn, personnummer og helseopplysninger.

Enkelte opplysninger vil ikke nødvendigvis kunne identifisere en person alene, men kan gjøre det dersom de kombineres med andre opplysninger. Dette er særlig relevant ved bruk av KI, fordi slike verktøy kan behandle og systematisere store mengder tekst. Flere små detaljer kan derfor gjøre det mulig å forstå hvem en tekst handler om, og dette kan få konsekvenser for personen det gjelder.

Hva bør du unngå å dele med KI?

Personopplysninger

Studenter bør ikke lime inn tekst som inneholder, eller kan inneholde, personopplysninger. Det gjelder ikke bare det som åpenbart er personopplysninger, men også beskrivelser som kan gjøre personer gjenkjennelige.

Tekst om andre personer

Intervjuer, refleksjonsnotater og oppgaver kan for eksempel inneholde detaljer om medstudenter, lærere og veiledere. Selv om navn er fjernet, kan andre detaljer gjøre det mulig å forstå hvem teksten handler om.

Konfidensielt materiale

Du bør også være forsiktig med å lime inn materiale som ikke er ment for deling. Dette kan for eksempel være interne dokumenter fra studiestedet, upubliserte forskningsdata, informasjon fra praksis eller materiale fra samarbeidsprosjekter.

Informasjon som ikke bør deles åpent

En enkel hovedregel er å ikke lime inn tekst du ikke ville delt åpent med andre. Trenger du hjelp fra KI, bør du heller skrive om teksten til en mer generell problemstilling og fjerne detaljer som kan identifisere personer eller andre involverte.

Hvordan bruke KI på en tryggere måte?

KI kan fortsatt være nyttig, selv om du ikke bør dele personopplysninger eller konfidensiell informasjon. En tryggere måte å bruke KI på er å stille mer generelle spørsmål, be om forklaringer, få hjelp til struktur, språk eller idéutvikling, og unngå å lime inn tekst som handler om konkrete personer.

  • Still generelle spørsmål i stedet for å lime inn tekst med personopplysninger.
  • Be om forklaringer, eksempler eller forslag til struktur.
  • Bruk KI til språk, idéutvikling og veiledning uten å dele sensitive detaljer.
  • Fjern navn, steder, små detaljer og andre opplysninger som kan gjøre personer gjenkjennelige.
  • Ikke lim inn interne dokumenter, praksisnotater, upubliserte forskningsdata eller annet konfidensielt materiale.

Kort oppsummert

Personvern ved bruk av KI handler først og fremst om å tenke gjennom hva du deler. Ikke legg inn personopplysninger, sensitive detaljer eller materiale som ikke er ment for deling. Bruk heller KI til generelle spørsmål, forklaringer og veiledning, og fjern detaljer som kan gjøre personer eller virksomheter gjenkjennelige.
